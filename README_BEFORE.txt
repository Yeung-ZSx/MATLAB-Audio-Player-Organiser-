# MATLAB-Audio-Player-Organiser-
Developing a MATLAB based Audio Player/Organiser that should be able to load and play audio in a few formats, 'wav , aiff , mp3 , mp4/aac , ogg'. With the means for organising the audio media into appropriate folders , playlist and including a Digital Signal Processing tool


To START and RUN the progrma, please run from "Start.m". And please be patience when running the program. 

To upload your own songs, just select from 'Add New Music' Tab.
